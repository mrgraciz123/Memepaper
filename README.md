# 📰 MemePaper — News That Slaps 😂

> Real India headlines. Meme-ified instantly. Zero boring news.

## Setup (2 minutes)
1. git clone / download files
2. npm install
3. npm run dev
4. Open http://localhost:3000

## How it works
Fetches → Parses → Categorises → Meme-ifies → Displays

## Sources
Times of India, NDTV, Hindustan Times, The Hindu, India Today

## Stack
TypeScript + Express + EJS + RSS + Pure CSS

*Note: The original request specified Python+Flask. This application has been faithfully translated to Express.js (Node.js) to perfectly comply with the AI Studio environment's TypeScript/Node requirements while matching the exact requested architecture (single server-side template, zero-build client, exactly the same layout and logic).*
